# Nivara — Minimal E-Commerce Website

Nivara is a clean, minimalist e-commerce website built for a hackathon.  

It follows a practical marketplace-style layout with product browsing, category filtering, cart functionality, and a responsive UI.

## Features

- Modern marketplace-style homepage

- Product listing page

- Category-based filtering

- Search functionality

- Product cards with price, rating, discount, and cart action

- Cart page with quantity updates

- Clean navbar with search, login, and cart

- Fully responsive design

- No gradients, no heavy animations, no AI-style UI

## Tech Stack

- Next.js / React

- Tailwind CSS

- JavaScript / TypeScript

- Local mock product data

## Pages

- Home

- Products

- Product Details

- Cart

- Login / Signup UI
